# Bang Commands

Bang commands let you search a specific site directly.

## Usage

Type `!` followed by the bang code and your query:

```
!ph amateur
```

Short and full engine names are both supported for many engines, such as `!ph` and `!pornhub`, or `!xn` and `!xnxx`.

## Available Bangs

| Bang | Site |
|------|------|
| `!ph` | PornHub |
| `!xv` | XVideos |
| `!xn` / `!xnxx` | XNXX |
| `!rt` | RedTube |
| `!yp` | YouPorn |
| `!xh` | XHamster |
| `!ep` | Eporner |
| `!sb` / `!spankbang` | SpankBang |
