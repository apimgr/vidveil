# Search

## Basic Search

Enter your search query in the search box and press Enter.

## Filtering Results

### By Engine

Select specific engines from the dropdown or use the `engines` parameter.

### By Duration

Use duration filters: short, medium, long

### By Quality

Filter by quality: HD, 4K

## Search Operators

| Operator | Example | Description |
|----------|---------|-------------|
| Quotes | `"exact phrase"` | Search for exact phrase |
| Minus | `-term` | Exclude term |
| Site | `site:pornhub` | Search specific site |
